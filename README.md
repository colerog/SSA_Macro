# **SSA Macro**
A macro for Bee Swarm Simulator focused on farming star signs.


## **Features**

### **Discord Integration**
SSA uses discord integration to send updates and any needed information via webhooks which have configurable settings in the macro.


### **Automically Installing Dependencies**
Automatically detects if you are missing dependencies and installs them, making it a more user friendly and easy program.


### **Automatic SSA Updatings**
Automatically will see if your version of SSA is up to date and if not it prompt you for an update and do everything automatically, including keeping your settings and data.
